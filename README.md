# موقع كرة القدم ⚽

موقع إلكتروني حديث لكرة القدم مطور باستخدام Python و Flask مع واجهة مستخدم جميلة ومتجاوبة.

## المميزات ✨

- 🏠 **الصفحة الرئيسية**: عرض شامل للفرق والمباريات والأخبار
- ⚽ **صفحة الفرق**: معلومات تفصيلية عن أشهر الفرق العالمية
- 📅 **صفحة المباريات**: جدول المباريات القادمة والمباشرة
- 📰 **صفحة الأخبار**: آخر أخبار كرة القدم مع نظام فلترة
- 🎨 **تصميم حديث**: واجهة مستخدم جميلة ومتجاوبة
- 🌐 **دعم اللغة العربية**: RTL والخطوط العربية
- 📱 **متجاوب**: يعمل على جميع الأجهزة
- ⚡ **سريع**: أداء محسن وتحميل سريع

## التقنيات المستخدمة 🛠️

- **Backend**: Python 3.x + Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **UI Framework**: Bootstrap 5 (RTL)
- **Icons**: Font Awesome 6
- **Fonts**: Google Fonts (Cairo)

## متطلبات التشغيل 📋

- Python 3.7+
- pip (مدير الحزم)

## طريقة التشغيل 🚀

1. **استنساخ المشروع**:
```bash
git clone <repository-url>
cd football-website
```

2. **إنشاء بيئة افتراضية** (اختياري ولكن مُوصى به):
```bash
python -m venv venv
source venv/bin/activate  # على Linux/Mac
# أو
venv\Scripts\activate     # على Windows
```

3. **تثبيت المتطلبات**:
```bash
pip install -r requirements.txt
```

4. **تشغيل الموقع**:
```bash
python app.py
```

5. **فتح المتصفح** والذهاب إلى:
```
http://localhost:5000
```

## API Endpoints 🔗

الموقع يوفر عدة endpoints لـ REST API:

- `GET /api/teams` - قائمة جميع الفرق
- `GET /api/matches` - قائمة جميع المباريات
- `GET /api/news` - قائمة جميع الأخبار

## البيانات 📊

الموقع يحتوي على بيانات عينة تشمل:

### الفرق:
- الريال مدريد
- برشلونة
- مانشستر يونايتد
- بايرن ميونخ
- باريس سان جيرمان
- يوفنتوس

### المباريات:
- مباريات من مختلف البطولات
- تواريخ وأوقات المباريات
- معلومات الفرق المتنافسة

### الأخبار:
- أخبار الانتقالات
- نتائج المباريات
- أخبار دوري أبطال أوروبا

## التخصيص 🎨

يمكن تخصيص الموقع عبر:

1. **إضافة بيانات جديدة** في `app.py`
2. **تعديل التصميم** في `static/css/style.css`
3. **إضافة صفحات جديدة** في `templates/`
4. **تعديل الألوان** في CSS Variables
